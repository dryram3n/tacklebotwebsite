Privacy Policy

Last Updated: April 10th, 2025

Introduction

This Privacy Policy outlines how TackleBot Discord Bot ("the Bot", "we", "us", "our") collects, uses, stores, and protects user data when you interact with our service. By using the Bot, you consent to the data practices described in this policy.

Data Collection

The Bot collects and processes the following data to provide its services:

1. User Information

  • Discord User IDs: For identifying users for game features, settings, moderation, channel creation, voting, whitelisting, bug reports, and command interactions.
  • Discord Server (Guild) IDs: For server-specific settings, moderation features, channel creation, logging configuration, and command interactions.
  • Command Interaction History: Basic information about which commands are used by which users and in which servers for functionality and improvement purposes.

2. Game and Feature Data

  • Fishing Game Data: Player statistics (level, XP, total catches, earnings, rarity counts, biggest fish, etc.), fish inventory, upgrade levels, research progress, challenge progress, preferred fishing locations, fishing company data. Stored persistently associated with your User ID.
  • Gamble Data: Gambling statistics (e.g., consecutive wins/losses). Stored persistently with your User ID.
  • Achievement Data: Progress and completion status for in-game achievements. Stored persistently with your User ID.

3. Moderation and Configuration Data

We store configuration settings for moderation features, including moderator lists and whitelists.

4. Content Data

Content processed for specific commands like /summary and /fun is mostly processed temporarily and not stored long-term by the Bot.

5. Vote Data

We track voting history via Top.gg to manage streaks and rewards.

6. Anti-Bot Data

We collect timing and interaction data to prevent automation and ensure fair play.

How We Use Your Data

Your data is used solely to:

  • Provide Core Bot Functionality: Execute commands, respond to interactions.
  • Manage Game Systems: Store and retrieve fishing progress, inventory, upgrades, currency, challenges, achievements, and company data.
  • Enable Moderation: Facilitate moderation commands and manage lists.
  • Generate Summaries: Process recent chat messages via Google Gemini for the /summary command upon user request.
  • Generate Fun Content: Process user-provided text and images via Popcat and Noodles APIs.
  • Manage User-Created Channels: Create and manage channels.
  • Track Voting: Record Top.gg votes and distribute rewards.
  • Provide Server Logging: Send logs of specified server events.
  • Prevent Abuse: Analyze fishing patterns to detect potential botting activity.
  • Manage Bug Reports: Store and manage bug reports.
  • Calculate Leaderboards: Determine weekly leaderboard winners.
  • Improve Bot Functionality: Analyze aggregated usage data.

Data Storage and Security

Duration of Storage

  • Game Data: Stored indefinitely until explicitly cleared by the user or data deletion request.
  • Moderator/Whitelist Data: Stored indefinitely until removed by a moderator/admin.
  • Vote Data: Stored indefinitely until data deletion is requested.
  • Temporary Data: Channel creator data, summary content, and anti-bot pattern data are stored temporarily.

Backup System

The Bot maintains automated backups of persistent user data, created regularly and retained for a limited time for disaster recovery.

Data Security

We implement reasonable technical and administrative security measures to protect your data. However, no online service can be 100% secure.

Third-Party Services

The Bot utilizes several third-party services that may process your data:

  • Discord API: Subject to Discord's Privacy Policy.
  • Google Generative AI (Gemini): Subject to Google's AI Principles and Privacy Policy.
  • Popcat API: Subject to Popcat API's data handling practices.
  • Noodles Wrapper (French Noodles API): Subject to the API provider's data handling practices.
  • Top.gg API & Webhooks: Subject to Top.gg's Terms of Service and Privacy Policy.

Your Rights

You have the right to:

  • Request access to your stored game data (e.g., using /fish stats).
  • Request deletion of your game data.
  • Disable vote reminders.
  • Opt out of data collection by removing the Bot from your server.

Data Deletion

Game data can be deleted upon request. Server administrators can remove users from moderation lists. Vote reminder preferences can be changed via buttons.

Changes to This Policy

We may update this Privacy Policy from time to time. We will notify users of significant changes.

Contact Information

For questions about this Privacy Policy or your data, please contact the development team, led by Thromblo, through the official Bot support server.
