Privacy Policy

Last Updated: June 9th, 2025

Introduction

This Privacy Policy outlines how TackleBot Discord Bot ("the Bot", "we", "us", "our") collects, uses, stores, and protects user data when you interact with our service. By using the Bot, you consent to the data practices described in this policy.

Data Collection

The Bot collects and processes the following data to provide its services:

1. User Information

  • Discord User IDs: Used for game features, settings, moderation, channel creation, voting, whitelisting, bug reports, and command interactions.
  • Discord Server (Guild) IDs: Used for server-specific settings, moderation tools, channel creation, logging configuration, and command interactions.
  • Command Interaction History: Basic information about which commands are used, by whom, and in which servers. Used to deliver functionality and improve features.

2. Game and Feature Data

  • Fishing Game Data: Player statistics (level, XP, total catches, earnings, rarity counts, biggest fish, etc.), fish inventory, upgrade levels (luck, speed, multicatch, inventory, value, explorer), research progress, challenge progress, preferred fishing locations, and fishing company data (boats, upgrades, earnings). Stored with your User ID.
  • Gamble Data: Gambling statistics such as consecutive wins/losses. Stored with your User ID.
  • Achievement Data: Progress and completion status for in-game achievements. Stored with your User ID.

3. Content Data (Mostly Temporary)

  • /summary: Recent public channel messages are sent to Google Generative AI (Gemini) to generate a summary. The Bot does not permanently store this content.
  • /fun (Text): User-provided text for commands such as achievement, changemymind, awkwardmonkey, burnpaper, drake, sadcat, oogway, biden, pikachu, mock, lulcat, opinion, imdb, and 8ball is sent to Popcat or Noodles APIs for processing and not stored long-term.
  • /fun (Images): Uploaded images or URLs for commands such as blur, wide, uglyupclose, communist, whowouldwin, ship, and opinion are sent to the relevant third-party API for processing.
  • /announce: Announcement content is processed temporarily.

4. Vote Data (Top.gg)

When you vote on Top.gg we receive your Discord User ID, timestamp of the vote, current vote streak, and vote reminder preferences. This data is stored until deletion is requested.

5. Moderation, Configuration, and Anti-Bot Data

Moderator lists, whitelists, logging settings, channel creator data, and timing/interaction data for anti-bot verification are stored to operate moderation and anti-abuse systems.

How We Use Your Data

Your data is used solely to:

  • Provide core bot functionality (commands, interactions, moderation, channel creation).
  • Manage game systems (fishing progress, inventory, upgrades, currency, challenges, achievements, companies, gambling features).
  • Generate summaries via Google Gemini when requested.
  • Generate fun content via Popcat and Noodles APIs for /fun commands.
  • Track votes, calculate streaks, award Voter Chests, and send reminders.
  • Store and manage bug reports.
  • Calculate weekly leaderboards and distribute rewards.
  • Improve Bot functionality by analyzing aggregated, anonymized usage data.

Data Storage and Security

Duration of Storage

  • Game Data: Stored indefinitely until cleared by the user or a deletion request.
  • Vote Data: Stored indefinitely until deletion is requested.
  • Bug Reports: Stored until resolved or closed.
  • Summary & Fun Content: Processed temporarily by third-party APIs; not stored permanently.
  • Channel Creator & Anti-Bot Data: Stored temporarily for operational needs.

Backup System

The Bot maintains automated backups of persistent user data (fishing stats, vote data, company data, etc.). Backups are created regularly (about every 30 minutes) and a limited number of recent backups (for example, the last three) are retained for disaster recovery.

Data Security

We implement reasonable technical and administrative security measures to protect your data from unauthorized access, alteration, disclosure, or destruction. No online service can be 100% secure.

Third-Party Services

The Bot utilizes third-party services that may process your data:

  • Discord API: Required for all bot interactions. Subject to Discord's Privacy Policy.
  • Google Generative AI (Gemini): Model gemini-2.0-flash used solely for the /summary command. Subject to Google's AI Principles and Privacy Policy.
  • Popcat API: Handles various /fun subcommands (e.g., fact, joke, imdb, sadcat, oogway, communist, whowouldwin, lulcat, opinion, ship, biden, pikachu, mock, 8ball).
  • Noodles Wrapper (French Noodles API): Handles other /fun subcommands (e.g., meme, achievement, changemymind, awkwardmonkey, blur, burnpaper, drake, wide, uglyupclose).
  • Top.gg API & Webhooks: Tracks votes, streaks, and rewards.

Your Rights

You have the right to:

  • Request access to stored game data (for example via /fish stats).
  • Request deletion of your game data (contact the development team or use available commands).
  • Disable vote reminders through provided buttons.
  • Opt out by removing the Bot from your server or discontinuing use.
  • Contact us with questions or concerns about your data.

Data Deletion

  • Fishing, gamble, achievement, and company data can be deleted upon request.
  • Server administrators can remove users from moderator lists, fishing whitelists, or logging configurations.
  • Users can delete channels they created (if permitted); admins/mods may also delete them. Channel creator data follows automated cleanup.
  • Vote reminder preferences can be updated via buttons; full vote history deletion requires contacting the development team.
  • Disabling logging stops future event logging and removes stored log channel configuration.

AI in Development and Service Enhancement

We may use AI tools and models from providers such as Google (Gemini) and Anthropic to aid in Bot development, maintenance, documentation, and operational support. These uses focus on non-user-specific content creation and service improvements. Any AI-powered feature that processes user data directly (like /summary) is described in the relevant sections above.

Changes to This Policy

We may update this Privacy Policy from time to time. Significant changes will be communicated by updating this page and potentially via in-bot announcements (for example, using /announce).

Legal Basis for Processing

  • Consent: Provided when you add the Bot or use commands that require data processing.
  • Legitimate Interest: Necessary to provide the Bot's core functionality, anti-abuse systems, and improvements.
  • Performance of a Contract (Implicit): We process data to provide the services you request by interacting with the Bot.

Contact Information
