Welcome to TackleBot! Here's how to get started:

  1. Add TackleBot to your server using the "Invite Bot" button.

Moderation Guide ğŸ›¡ï¸

TackleBot includes useful moderation features for your server:

Basic Commands

  • /kick [user] [reason] - Remove user from server
  • /ban [user] [reason] [days] - Ban user and optionally delete messages
  • /clear [amount] - Delete up to 100 recent messages

Managing Moderators

  • /moderator add [user] - Add a new moderator
  • /moderator remove [user] - Remove existing moderator
  • Only server owners can manage moderators

Channel Management

  • /createchannel [name] type:[text/voice] - Create a text or voice channel
  • /createchannel [name] private:true member1:[user] - Create a private channel
  • /deletechannel - Delete the channel you're currently in
  • Limit: 1 text and 1 voice channel per user

Note: Users can only delete channels they created or if they have server management permissions. Moderators and server owners can delete any channel.
