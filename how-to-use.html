How to Use TackleBot ‚ùì

Welcome! TackleBot uses Discord's slash commands. Just type / in your chat bar and select a TackleBot command. Click on any command group below to see details!

Basic information and help commands.

  • /info: Shows the bot's current version, uptime, latest updates, and links.
  • /help [section]: Provides help information.
    • /help commands: Lists all available command groups.
    • /help fishing: Detailed guide on the fishing system.
    • /help settings: Information about bot configuration and permissions.
    • /help moderation: Guide for moderation commands.
    • /help fun: Guide for fun/entertainment commands.
    • /help legal: Information on Terms of Service and Privacy.
    • /help donate: Information on how to support the bot.
  • /donate: Shows ways to support TackleBot's development.
  • /vote: Provides a link to vote for TackleBot on Top.gg and explains rewards.
  • /tos: Displays the bot's Terms of Service.

The core fishing gameplay commands.

  • /fish cast [location]: Cast your fishing rod! Optionally specify a location (e.g., pond, river) if unlocked.
  • /fish inventory: View the fish you've caught. Use the buttons to navigate pages.
  • /fish sell [index]: Sell a specific fish from your inventory using its number (from /fish inventory).
  • /fish sellall: Sell all fish currently in your inventory.
  • /fish shop: View available upgrades for purchase (Lures, Rods, etc.).
  • /fish buy [upgrade]: Purchase an upgrade from the shop.
  • /fish stats [user]: View your own or another user's fishing statistics (level, XP, catches, etc.).
  • /fish challenge: View your current daily/seasonal fishing challenges and progress.
  • /fish index: Browse the complete catalog of discoverable fish species.
  • /fish company: Manage your fishing company (passive income feature).

Entertainment and image manipulation commands.

  • /fun meme: Fetches a random meme.
  • /fun achievement [text]: Creates a Minecraft-style achievement image.
  • /fun changemymind [text]: Creates a "Change My Mind" meme image.
  • /fun drake [text1] [text2]: Creates a Drake meme image.
  • /fun blur [image/user]: Blurs the provided image or user's avatar.
  • /fun wide [image/user]: Makes the provided image or user's avatar wide.
  • /fun communist [image/user]: Adds a communist filter to an image/avatar.
  • /fun fact: Displays a random interesting fact.
  • /fun joke: Tells a random joke.
  • /fun imdb [title]: Searches for movie/show information on IMDb.
  • /fun lulcat [text]: Converts text to LOLCAT speak.
  • /fun showerthought: Displays a random shower thought.
  • /fun ship [user1] [user2]: "Ships" two users and shows a compatibility percentage.
  • /fun opinion [text] [image/user]: Creates an "opinion" meme.
  • /fun whowouldwin [user1] [user2]: Creates a "Who Would Win" meme.
  • /fun mock [text]: Converts text to MoCkInG CaSe.

Commands for server management (require appropriate permissions).

  • /kick [user] [reason]: Removes a user from the server. Requires Kick Members permission.
  • /ban [user] [reason] [days]: Bans a user from the server. Optionally delete messages from the past specified days. Requires Ban Members permission.
  • /clear [amount]: Deletes a specified number of recent messages (up to 100). Requires Manage Messages permission.
  • /moderator add [user]: Grants a user permission to use TackleBot's moderation commands within the bot. Only usable by the server owner.
  • /moderator remove [user]: Revokes a user's permission to use TackleBot's moderation commands. Only usable by the server owner.
  • /log setup [channel]: Sets up a channel for logging server events (message deletes, member joins/leaves). Requires Manage Server permission.
  • /log disable: Disables server event logging. Requires Manage Server permission.
  • /fishwhitelist add [user]: Adds a user to the fishing anti-bot whitelist (fewer checks). Requires Manage Server permission or Bot Moderator role.
  • /fishwhitelist remove [user]: Removes a user from the fishing whitelist. Requires Manage Server permission or Bot Moderator role.

Commands for creating and deleting channels.

  • /createchannel [name] [type] [private] [member1...]: Creates a new channel.
    • name: The desired channel name.
    • type: (Optional) Choose 'text' (default) or 'voice'.
    • private: (Optional) Set to 'true' to make it private.
    • member1, member2, etc.: (Optional, if private) Mention users to grant access.
    • Requires Manage Channels permission, but users can create a limited number for themselves.
  • /deletechannel: Deletes the channel this command is used in. Requires Manage Channels permission, OR the user must be the original creator of the channel (via /createchannel).

Commands related to the Discord server itself.

  • /server stats: Displays detailed statistics about the current server (member count, channel count, roles, creation date, etc.).

Unique utility and interaction commands.

  • /rizz [message]: Summons the legendary Ronald Rizz (if he's in the server) with an optional message.
  • /summary [channel] [messages]: Generates a humorous summary of recent messages in the current or a specified channel using AI.
  • /bug report: Opens a modal to report a bug or issue with the bot.
  • /bug view [id]: View the status or details of a specific bug report (Admin/Dev only).
  • /bug respond [id]: Respond to a bug report (Admin/Dev only).
  • /bug close [id]: Close a bug report (Admin/Dev only).
