Privacy Policy 🔒

Introduction

This Privacy Policy outlines how TackleBot Discord Bot ("the Bot", "we", "us", "our") collects, uses, stores, and protects user data when you interact with our service. By using the Bot, you consent to the data practices described in this policy.

Data Collection

Information We Collect

The Bot collects and processes the following data to provide its services:

1. User Information:

  • Discord User IDs: For identifying users for game features, settings, moderation, and command interactions.
  • Discord Server (Guild) IDs: For server-specific settings, moderation features, and command interactions.
  • Command Interaction History: Basic information about which commands are used for functionality and improvement purposes.

2. Game and Feature Data:

  • Fishing Game Data: Player statistics (level, XP, total catches, earnings), fish inventory, upgrade levels, research progress, challenge progress, fishing company data. Stored persistently associated with your User ID.
  • Achievement Data: Progress and completion status for in-game achievements. Stored persistently with your User ID.

3. Content Data (Processed, Not Stored Long-Term):

  • /summary Command: Content of recent public messages in the channel where the command is used. This content is sent to Google Generative AI (Gemini) for processing but is not permanently stored by the Bot.
  • /fun Commands: Text or images provided by users for various fun commands. This data is sent to third-party APIs for processing and is not permanently stored by the Bot.

4. Vote Data (via Top.gg):

  • When you vote on Top.gg, we receive your Discord User ID via a webhook.
  • We store your User ID, the timestamp of your last vote, your current vote streak, and your preference for vote reminders.

How We Use Your Data

Your data is used solely to:

  • Provide core Bot functionality and execute commands
  • Manage game systems and store fishing progress
  • Generate summaries via Google Gemini for the /summary command
  • Generate fun content via third-party APIs for /fun commands
  • Track Top.gg votes and calculate streaks
  • Improve Bot functionality through anonymized usage analysis

Data Storage and Security

Duration of Storage

  • Game Data: Stored indefinitely until explicitly cleared by the user or upon data deletion request
  • Vote Data: Stored indefinitely until data deletion is requested
  • Summary & Fun Command Content: Processed temporarily by third-party APIs; not stored permanently by the Bot

Backup System

  • The Bot maintains automated backups of persistent user data
  • Backups are created regularly (approximately every 30 minutes)
  • A limited number of recent backups are retained for disaster recovery purposes

Data Security

We implement reasonable technical and administrative security measures to protect your data from unauthorized access, alteration, disclosure, or destruction. However, no online service can be 100% secure.

Third-Party Services

The Bot utilizes several third-party services that may process your data:

  1. Discord API: Used for all Bot interactions within Discord. Subject to Discord's Privacy Policy.
  2. Google Generative AI (Gemini): Used for chat summarization. Subject to Google's Privacy Policy.
  3. Popcat API: Used for various fun commands. Subject to Popcat's terms.
  4. Noodles Wrapper (French Noodles API): Used for various fun commands. Subject to French Noodles' terms.
  5. Top.gg API & Webhooks: Used for vote tracking. Subject to Top.gg's Privacy Policy.

Your Rights

You have the right to:

  • Request access to your stored game data (e.g., using /fish stats)
  • Request deletion of your game data (contact the development team)
  • Disable vote reminders via the button in the reminder DM
  • Opt out of data collection by removing the Bot from your server or discontinuing its use
  • Contact us with questions or concerns about your data

Data Deletion

  • Game data can be deleted upon request to the development team
  • Vote reminder preferences can be changed using the provided buttons
  • Full vote history deletion requires contacting the development team

AI in Development and Service Enhancement

To aid in the development, maintenance, and improvement of TackleBot, we leverage artificial intelligence (AI) tools and models from various providers, which may include Google (e.g., Gemini models) and Anthropic. These technologies are primarily used for development and enhancement purposes.

Any AI-powered features that directly process user data, such as the /summary command's use of Google Gemini, are explicitly detailed within the 'Data Collection' and 'Third-Party Services' sections of this policy.

Changes to This Policy

We may update this Privacy Policy from time to time. We will notify users of any significant changes by updating the "Last Updated" date of this policy, and potentially through in-Bot announcements.

Contact Information

For questions about this Privacy Policy or your data, please contact the development team, led by Thromblo, through the Discord support server.

Legal Basis for Processing

We process your data based on your consent when using the Bot and for our legitimate interests in providing and improving our services.
